Low-End PC

This PC is tailored for the low-end user looking to use their PC mainly for work or web browsing although this PC will still be able to handle 50FPS in some games such as Fortnite, CS GO, Minecraft, and other games in this league. So if you are looking for a cheaper PC that doesn’t cost a fortune and can do exactly everything you need it to do then this is the one for you, but if it doesn’t cross off your list then consider checking out a different template.


Parts in This Build

  • AMD Ryzen 3 3200G
  • Gigabyte B450M DS3H
  • Patriot Viper Steel Series DDR4 8GB (2 x 4GB) 3200MHz Memory Kit
  • Kingston 240GB A400 SSD 2.5'' SATA 7MM 2.5-Inch SA400S37/240G
  • Corsair Semi-Modular ATX CX450M Power Supply
  • CORSAIR CARBIDE SPEC-05 Mid-Tower Gaming Case

Also, feel free to change some of the parts in this build to your liking if you want, you could even throw in a graphics card to get more frames in gaming.
